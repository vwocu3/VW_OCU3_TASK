Exit codes of the tool that indicate a failure but should be ignored, thus causing the build to proceed.
Ignored exit codes may be specified as
  • a single exit code, e.g. 2,
  • a comma separated list, e.g. 2,21,50,
  • a range, e.g. 50-70,
  • a combination of the above,
  • the empty string to disable this feature.
