CMakeÂ´s buildscript generator to use (e.g. Unix Makefiles). Leave empty to let CMake choose a generator.
Possible generators include:
  • Borland Makefiles
  • MSYS Makefiles
  • MinGW Makefiles
  • NMake Makefiles
  • NMake Makefiles JOM
  • Ninja
  • Unix Makefiles
  • Watcom WMake
  • Visual Studio 10 2010 (CMAKE_BUILD_TOOL not exposed, generation failure workaround)
  • Visual Studio 11 2012 (CMAKE_BUILD_TOOL not exposed, generation failure workaround)
  • Visual Studio 12 2013 (CMAKE_BUILD_TOOL not exposed, generation failure workaround)
  • Xcode (CMAKE_BUILD_TOOL not exposed, generation failure workaround)

HINT: This field does not require quotes if the generator name contains spaces.

The build tool corresponding to the specified script generator is exposed in the CMAKE_BUILD_TOOL build variable, except for the Visual Studio and Xcode generators (the actual value of CMAKE_BUILD_TOOL is retrieved from the generated CMake cache file, but these generators do not set CMAKE_MAKE_PROGRAM at buildscript generation time).
